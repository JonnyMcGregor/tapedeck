# Ascii DAW

## Features
- Audio input
- Audio output
- Record one track at a time (consider recording multiple at a time)
- Play back tracks while recording
- Mute and solo
- Save and load
- Add and remove and move around audio clips
- Add and remove tracks, name tracks
- Colour coding for tracks
